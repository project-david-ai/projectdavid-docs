\section{Supervisor State Machine}

\[
Idle \rightarrow StrategyUpdate
\]
\[
StrategyUpdate \rightarrow Delegating
\]
\[
Delegating \rightarrow AwaitingWorker
\]
\[
AwaitingWorker \rightarrow Evaluating
\]
\[
Evaluating \rightarrow Delegating \quad \textit{(if incomplete)}
\]
\[
Evaluating \rightarrow Synthesizing \quad \textit{(if complete)}
\]
\[
Synthesizing \rightarrow Terminated
\]
